Proxy Scholae

Scholae fournit un accès Internet libre pour tous ses visiteurs. La loi nous oblige cependant à filtrer cet accès pour garder un suivi complet de toutes les connexions.

Si vous souhaitez accéder à Internet, il vous suffit d'installer le certificat SSL de l'entreprise. Cette page contient des instructions pour les systèmes et les navigateurs web les plus courants.

Sélectionnez votre plateforme en cliquant sur le logo correspondant.

Microsoft Windows

Sélectionnez votre navigateur web dans la liste.

Mozilla Firefox pour Windows

  1. Téléchargez le certificat.
  2. Dans la fenêtre de téléchargement, choisissez Enregistrer le fichier.
  3. Rangez le fichier dans vos Téléchargements.
  4. Si vous ne voyez pas la barre de menus de Firefox, faites un clic droit sur les trois traits horizontaux Ouvrir le menu en haut à droite de la fenêtre et cochez Barre de menus.
  5. Ouvrez Outils > Options.
  6. Cliquez sur l'onglet Vie privée et sécurité et descendez à la section Certificats.
  7. Si vous avez une ancienne version de Firefox, la section Certificats se trouve dans l'onglet Avancé.
  8. Cliquez sur Afficher les certificats, puis sur Importer.
  9. Importez le certificat téléchargé et cliquez sur Ouvrir.
  10. Cochez les trois options Confirmer cette AC... et terminez en cliquant sur OK.

Google Chrome pour Windows

  1. Téléchargez le certificat.
  2. Ouvrez le menu principal de Chrome en cliquant sur le bouton Personnaliser et contrôler Google Chrome symbolisé par trois petits points verticaux en haut à droite de la fenêtre.
  3. Allez dans Paramètres.
  4. Descendez sur la page et cliquez sur Paramètres avancés.
  5. Cliquez sur Gérer les certificats.
  6. Cliquez sur Importer.
  7. L'Assistant Importation de Certificats se lance. Cliquez sur Suivant.
  8. Repérez le champ Nom du fichier et cliquez sur Parcourir.
  9. Dans le menu déroulant en bas à droite du sélecteur de fichiers, choisissez Tous les fichiers (*.*).
  10. Sélectionnez le certificat proxy.scholae.lan et cliquez sur Ouvrir, puis sur Suivant.
  11. Dans l'option Magasin de certificats, cliquez sur Parcourir et choisissez Autorités de certification racines de confiance. Cliquez sur Suivant.
  12. Cliquez sur Terminer et confirmez l'installation du certificat.

Internet Explorer pour Windows

  1. Téléchargez le certificat en effectuant un clic droit sur ce lien et en choisissant Enregistrer la cible sous... dans le menu contextuel.
  2. Enregistrez le fichier proxy.scholae.lan dans vos Téléchargements.
  3. Dans le menu d'Internet Explorer en haut à droite de la fenêtre, ouvrez Outils > Options Internet.
  4. Dans l'onglet Contenu, cliquez sur le bouton Certificats.
  5. Cliquez sur Importer, ce qui lance l'Assistant Importation de Certificats. Cliquez sur Suivant.
  6. Repérez le champ Nom du fichier et cliquez sur Parcourir.
  7. Dans le menu déroulant en bas à droite du sélecteur de fichiers, choisissez Tous les fichiers (*.*).
  8. Sélectionnez le certificat proxy.scholae.lan et cliquez sur Ouvrir, puis sur Suivant.
  9. Dans l'option Magasin de certificats, cliquez sur Parcourir et choisissez Autorités de certification racines de confiance. Cliquez sur Suivant.
  10. Cliquez sur Terminer et confirmez l'installation du certificat.

Mac OS X

Mac OS X stocke les certificats au niveau du système. On va donc utiliser le navigateur Safari pour récupérer le certificat. Une fois installé, on pourra naviguer sur le web avec Safari, Firefox Chrome ou n'importe quel autre navigateur.

  1. Téléchargez le certificat.
  2. Récupérer le fichier proxy.scholae.lan.der dans les Téléchargements.
  3. Double-cliquer sur le certificat téléchargé, ce qui ouvre le Trousseau d'accès.
  4. Repérer le certificat dans le Trousseau d'accès et double-cliquer dessus pour ouvrir les préférences du certificat.
  5. Déplier la section Se fier.
  6. Repérer la section Lors de l'utilisation de ce certificat et sélectionner l'option Toujours approuver dans le menu déroulant.
  7. Fermer la fenêtre et fournir le mot de passe du système pour prendre en compte les modifications.

Linux

Sélectionnez votre navigateur web dans la liste.

Mozilla Firefox pour Linux

  1. Cliquez ici pour importer le certificat directement.
  2. Cochez les trois options Confirmer cette AC... et terminez en cliquant sur OK.

Google Chrome pour Linux

  1. Téléchargez le certificat.
  2. Ouvrez le menu principal de Chrome en cliquant sur le bouton Personnaliser et contrôler Google Chrome symbolisé par trois petits points verticaux en haut à droite de la fenêtre.
  3. Allez dans Paramètres.
  4. Descendez sur la page et cliquez sur Paramètres avancés.
  5. Cliquez sur Gérer les certificats.
  6. Ouvrez l'onglet Autorités.
  7. Cliquez sur Importer.
  8. Importez le fichier proxy.scholae.lan.der depuis vos Téléchargements.
  9. Cochez les trois options Considérer ce certificat comme fiable... et terminez en cliquant sur OK.

Konqueror pour Linux

  1. Téléchargez le certificat.
  2. Enregistrez le fichier proxy.scholae.lan.der dans vos Téléchargements.
  3. Ouvrez Menu KDE > Configuration du système > Préférences SSL.
  4. Cliquez sur Ajouter.
  5. Sélectionnez le certificat proxy.scholae.lan.der dans vos Téléchargements.
  6. Confirmez en cliquant sur Appliquer.
  7. Le certificat apparaît tout en bas dans la liste, dans les Certificats ajoutés par l'utilisateur.

Mobiles et tablettes Android

Chrome est le principal navigateur du système Android.

Chrome pour Android

  1. Récupérez le certificat en sélectionnant l'option Télécharger le lien.
  2. Ouvrez le fichier proxy.scholae.lan.pem.
  3. Saisissez le mot de passe du système.
  4. Donnez un nom au certificat, par exemple Proxy.
  5. Sélectionnez l'option Utilisé pour VPN et applications.

iPhones et iPads

Safari est le principal navigateur du système iOS.

Safari pour iOS

  1. Démarrez l'installaton du certificat en cliquant dessus.
  2. Autorisez l'ouverture des Réglages en cliquant sur Autoriser.
  3. Cliquez sur Installer pour installer le certificat.
  4. Fournissez le mot de passe du système.
  5. Cliquez encore une fois sur Installer.
  6. Confirmez par OK.
  7. Quittez Safari et relancez-le.
